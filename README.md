> The quickest way to run [vuejs] client side (inspired by [next.js], [nuxt.js] and  [vbuild]).

## Quickstart

```
$ yarn global add vues
$ vues init project
$ cd project
$ yarn
$ vues
```
Done!!!  You're now running [vuejs].
